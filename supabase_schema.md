# Supabase Database Schema

This document outlines the current database schema for the Kanti-mart Supabase project, including tables, columns, data types, and foreign key relationships for the `public` schema.

## Tables

### 1. `categories`
Stores product categories available in the store.
- **RLS Enabled:** Yes
- **Primary Key:** `id`

| Column | Type | Attributes | Description |
| :--- | :--- | :--- | :--- |
| `id` | `bigint` (int8) | `identity`, `updatable` | Unique identifier (generated by default). |
| `name` | `text` | `updatable` | The name of the category (e.g., "Groceries"). |
| `icon_name` | `text` | `nullable`, `updatable` | Optional icon identifier. |

---

### 2. `products`
Stores the e-commerce products.
- **RLS Enabled:** Yes
- **Primary Key:** `id`
- **Foreign Keys:**
  - `category_id` references `categories.id`

| Column | Type | Attributes | Description |
| :--- | :--- | :--- | :--- |
| `id` | `bigint` (int8) | `identity`, `updatable` | Unique identifier (generated by default). |
| `created_at` | `timestamptz` | `nullable`, `updatable` | Timestamp of creation (default: `now()`). |
| `name` | `text` | `updatable` | Product name. |
| `price` | `numeric` | `updatable` | Price of the product. |
| `weight_volume` | `text` | `nullable`, `updatable` | Measurement (e.g., "500g", "1L"). |
| `image_url` | `text` | `nullable`, `updatable` | Link to product image in storage bucket. |
| `category_id` | `bigint` (int8) | `nullable`, `updatable` | ID of the category this product belongs to. |
| `stock_count` | `integer` (int4) | `nullable`, `updatable` | Amount in stock (default: `0`). |

---

### 3. `profiles`
Stores extended user profile data, automatically linked to Supabase Auth.
- **RLS Enabled:** Yes
- **Primary Key:** `id`
- **Foreign Keys:**
  - `id` references `auth.users.id`

| Column | Type | Attributes | Description |
| :--- | :--- | :--- | :--- |
| `id` | `uuid` | `updatable` | Primary key linked directly to Auth user ID. |
| `full_name` | `text` | `nullable`, `updatable` | Full name of the user. |
| `avatar_url` | `text` | `nullable`, `updatable` | Link to the user's avatar image. |
